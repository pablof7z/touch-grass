# Chief Of Staff README Proposals

These are independent public model-card directions for `chief-of-staff`.

Each proposal is self-contained and includes:

- a GitHub-ready `README.md`;
- a real photo banner asset;
- banner source and license notes.

The canonical runtime profile remains `../agent.yaml`. These proposal READMEs
are candidate public-facing model-card bodies for
`agents/chief-of-staff/README.md`.

## Proposals

| Direction | Folder | Core angle |
| --- | --- | --- |
| Operating Picture | [`01-operating-picture`](01-operating-picture/) | Know what moved, what matters, what is blocked, and what needs the user. |
| Attention Firewall | [`02-attention-firewall`](02-attention-firewall/) | Stop interrupting the user for low-stakes agent-system decisions. |
| Workflow Memory | [`03-workflow-memory`](03-workflow-memory/) | Turn repeated user asks into learned operating workflows. |
| Human Judgment Router | [`04-human-judgment-router`](04-human-judgment-router/) | Let agents continue until a real authority or judgment boundary appears. |
| GitHub As The Office | [`05-github-as-the-office`](05-github-as-the-office/) | Keep operating state in durable GitHub artifacts instead of chat drift. |
