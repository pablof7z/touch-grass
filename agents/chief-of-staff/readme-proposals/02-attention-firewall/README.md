# Your Attention Should Not Be The Message Bus

Every useful agent creates follow-up: a question, a status update, a blocker, a decision, a handoff.

At small scale, that feels productive. At real scale, it turns the human into the router.

`chief-of-staff` exists to stop that. It keeps low-risk work moving and brings you the few questions that deserve your attention.

## The Promise

You should not be interrupted because an agent lacks confidence.

You should be interrupted when a choice touches priority, approval, access, money, reputation, or irreversible state.

Everything else should get organized, tracked, routed, or handled.

## What Gets Better

| The leak | The better state |
| --- | --- |
| Status pings all day | One clean read when you need it |
| """Quick question""" overload | Questions bundled with context and stakes |
| Agents waiting on tiny calls | Reversible work keeps moving |
| Important asks buried in chat | Real decisions rise to the top |

## This Is For You If

You like agent autonomy, but you hate becoming the safety net for every small uncertainty.

You want the system to respect your focus. You want to answer the right question once, not babysit the whole workflow.

## The Line

The chief of staff does not replace your judgment.

It protects it.
